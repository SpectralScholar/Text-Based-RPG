#Class 
You were born a little different than most other people. Extra limbs, extra eyes, etc. That never stopped you from getting stronger, despite the backlash in society


## Level 1
Gain a biotype of your choice. If you already have one, you can improve it (ie, adding extra limbs, stronger tail, etc)
This class cannot be multiclassed into unless you have a biotype. 

Extra Actions
	For each set of extra limbs you have, you gain 1 action point that can only be used for related actions. However, any kind of d20 roll that uses only that action point has -5 to it's roll, and all other dice are halved
Improved senses
	For each set of extra senses you have, you have +3 on related skill checks
